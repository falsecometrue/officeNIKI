# 02: HTML/CSS/IMG 解析

このフォルダは現状の解析をまとめるための場所です。

- `html_css_img_analysis.py` : HTML/CSS/画像/JS を解析し、構造化データを出力します。
- `analysis_output.json` : JSON 形式の解析結果。
- `analysis_output.md` : Markdown 形式の解析レポート。

入力ファイルは `../../30.test/testData/Excel_sampleData/シート1.html` です。

## 目的

- `Excel_sampleData` の HTML 出力を詳細に解析する
- テーブル構造、画像埋め込み、JS によるレイアウト制御、CSS クラス利用状況を確認する
- これを元に Markdown や draw.io への変換方針を検証する
