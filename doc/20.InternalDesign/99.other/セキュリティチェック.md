# セキュリティチェック観点表

## 対象

- 対象拡張機能: `office-to-mdrow` ディレクトリ配下のVS Code拡張機能
- package名: `office-to-mdraw`
- 対象バージョン: `package.json` の `version`
- 主な機能:
  - `.xlsx` から `.drawio` への変換
  - `.xlsx` から Markdown と `resources/` への変換
  - `.docx` から Markdown と `resources/` への変換
  - `.pptx` から Marp Markdown と `slide/*.drawio.svg` への変換
- データ処理方針: 選択されたOfficeファイルをローカルで読み取り、変換結果を元ファイルと同じフォルダ配下に出力する
